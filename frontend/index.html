About Eye AI
Professional retinal screening system

Eye AI is a multi-agent agentic AI system for diabetic retinopathy screening. It uses an EfficientNet-B3 vision model to classify retinal images into 5 DR severity grades, then passes results through a 5-agent pipeline (image analysis, diagnosis, trend, alert, report) to produce a full clinical report.

Vision model
EfficientNet-B3 (timm)
Agent framework
LangGraph + GPT-4o-mini
Backend
FastAPI + SQLite

DR grades 0–4, Glaucoma, AMD
