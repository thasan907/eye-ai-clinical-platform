🤖 5-Agent pipeline
1
🔬 Image Analysis Agent
Interprets raw model output into clinical language. Identifies microaneurysms, haemorrhages, exudates, and neovascularisation from the confidence scores.
2
🩺 Diagnosis Agent
Combines vision model output with patient HbA1c, age, and diabetes type. Generates a personalised clinical assessment with risk factors.
3
📈 Trend Agent
Compares current scan to patient's previous visits. Flags progression, improvement, or stability. Recommends monitoring frequency.
4
🚨 Alert Agent
Rule-based escalation engine. Triggers urgent flag for Severity ≥ 3. Generates emergency alert message with patient ID and recommended action.
5
📄 Report Agent
Assembles all agent outputs into a plain-English patient summary and a structured clinical recommendation for the treating physician.
